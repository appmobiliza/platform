# SUMMARY.md — Projeto Mobiliza

## Resumo Executivo

O **Mobiliza** é um sistema digital de mobilidade assistida para estudantes com deficiência (PcD) da UFAL, desenvolvido como projeto acadêmico em parceria com o Núcleo de Acessibilidade (NAC). O projeto substitui o fluxo manual atual — solicitação via WhatsApp e registro em planilhas preenchidas retroativamente — por uma plataforma integrada composta por **aplicativo mobile** (React Native), **dashboard web** (Next.js 16) e **backend** (Node.js + tRPC + PostgreSQL via Drizzle ORM).

**Problema central:** O MobiUFAL opera sem infraestrutura tecnológica dedicada, comprometendo privacidade dos estudantes, rastreabilidade dos atendimentos e capacidade de gestão do NAC.

**Solução:** Plataforma com solicitação privada de deslocamento, matching por disponibilidade de bolsistas, registro automático de atendimentos e dashboard em tempo real para coordenação.

**Stack:** TypeScript strict, Turborepo monorepo, tRPC v11, Next.js 16, React 19, Drizzle ORM, PostgreSQL/Neon, TailwindCSS v4, WebSockets.

**Entrega:** Protótipo funcional em ambiente de desenvolvimento — Julio 2026.
